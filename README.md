# KISU

## Name : Kisu 🐶

> **Kisu** is a visual radio that showcases anime openings and endings 🎌.

## Features:

1. [x] **Must have:**
    1. [x] Create a player
    2. [x] Define some database
    3. [x] Show current video informations (title, author, category)
    4. [x] Autoplay

2. [ ] **Have:**
    1. [x] Sort / Filter
    2. [x] Random
    3. [x] More informations
    4. [ ] Share current video

3. [ ] **Best Case:**
    1. [ ] DDL video
    2. [ ] Account
    3. [ ] Likes
    4. [ ] Comments
    5. [ ] Vues
    6. [ ] Blind test

## Vocabulary :

* `shojo` : Shojo
* `shonen` : Shonen
* `seinen` : Seinen
* `op` : Opening
* `en` : Ending
