## Name : Kisu

## Pitch

**Kisu** is a visual radio that showcases anime openings and endings.

## Features : Must have

1. Create a player
2. Define some database
3. Show current video informations (title, author, category)

## Features : Have

1. Sort / Filter
2. Random
3. More informations
4. Share current video

## Features : Best Case

1. DDL video
2. Account
3. Likes
4. Comments
5. Vues
